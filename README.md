# Sensortag-Websocket
Example of feeding sensortag data into websocket

## Install

	yarn install
	gulp
	node app.js

## Raspberry Pi 3 Install

- Use the `linux` branch

## OSX Install

- Use the `master` branch
- Tested on OS X EI Capitan 10.11.6
- `xpc-connection` and `bplist-parser` must be installed.
